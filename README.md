# Bakswitch

The backup switcher

```
Program to swap original and backup file

  --help            -h           show this message
  
USAGE
    $ bakswitch <extension>
    if extension is not specified, it is default to "bak"
        
EXAMPLE
    $ bakswitch backup
    will swap file named "nginx.conf.backup" to "nginx.conf" and "nginx.conf" to "nginx.conf.backup"
```
